Colors in CSS

Can be denoted a number of different ways

  • Color names color: red;
  • HTML5 color names color: blue;
  • Hexadecimal code: color: #00ff00;
  • RGB color: rgb(0,0,255);

ID vs Class

Different elements can be given an id or a class to apply styling to from the css file. An id should only be given to a single element while classes can be used by multiple elements to apply the same styling.

Box model

The box model is made up of four sections

  • Content: Made up from what your content actuially is.
  • Padding: The space between the content element & its border.
  • Border: Not always used but can be set to give further spacing
  • Margin: The space between the border and the adjacent elements

Padding, border and margins all have a left, right, top and bottom parameter which can be altered

Applying margin and padding

When applying margin and padding widths they can be written in long hand or short hand an example oif each can be found below.

Long Hand

p{

margin-top: 5px;

margin-right: 10px;

margin-bottom: 5px;

margin-left: 10px;

}

Short Hand

p{

(margin: TOP RIGHT BOTTOM LEFT;)

margin: 5px 10px 5px 10px;

}

Positioning in CSS

Typed of positioning

  • Static: If not stated static is default and will render elements in order of the document.
  • Relative: relative will shift the element relative to its normal position, by using commands like top, left etc.
  • Absolute: Allows positional targeting of an element within a relative structure
  • Fixed: will create a persistant element on the browser regardless of where we scroll, often used for nav bars
  • Initial: Sets the property to its default value
  • Inherit: will inherit what ever properties belong to the parent element
